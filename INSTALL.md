# LookML Framework - How to install

## How to copy the template to your Looker project

1. In Looker, switch on `Development Mode`
1. In the list of projects press `Configure new Model`
![Configure a new model in Looker](https://www.onesecondbefore.com/images/looker/looker-configure-a-model.png "Configure a new model in Looker")
1. Enter a `model name`, `project name` and which connection is allowd to connect
1. Press `Save`. This should bring you to the Project overview.
1. Search for the project in the list and press `Add LookML` (on the right)
![Add LookML by cloning Github project](https://www.onesecondbefore.com/images/looker/looker-create-new-project-from-github-clone.png "Add LookML by cloning Github project")
1. Give your project a name, choose `Clone Public Git Repository` as your starting point. Use the Git Repository URL: `https://github.com/onesecondbefore/lookml-framework.git`
1. Press `Create Project` (can take a bit, Looker now copies the framework)
1. Looker will now redirect you to the newly created project
1. In the left bar, under `Project Settings`, choose `Reset Git Connection`.
![Reset Git Connection](https://www.onesecondbefore.com/images/looker/looker-project-reset-git-connection.png "Reset Git Connection")
1. Configure Looker to use your own Git repository.
1. Enjoy!
